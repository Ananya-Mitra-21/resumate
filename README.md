## Resume Builder ‚Äì React + jsPDF + html2canvas(Day 4-5)

## üöÄ Features

- **Real-time Preview**  
  Resume updates instantly as you type.

  Export resumes to a properly scaled **A4 PDF** using `html2canvas` and `jsPDF`.

  - Experience
